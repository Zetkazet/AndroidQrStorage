# AndroidQrStorage
Application for creating basic qr based storage management

App special for storage recap with QR code. 

QR code to used the scanner. : 

pid (product id) pname (product name)

Feature :
(d) = done
(wp) = work in progress

Android : 

- Login (d)
- logout (d)
- QR scanner (done)
- Stock in (done)
- Stock out (done)
- Stock View (wp)
- Stock Register (wp)

Web :

Backend :
- stock in (D)
- Stock out (d)
- Add user (wp)
- add item (wp)

Front end:

- all (wp)
